# 1.2.1

**Released:** 2026-03-29

## Summary

- Fixed iOS app target `FRAMEWORK_SEARCH_PATHS` misconfiguration
- Prevented misquoted `$(inherited)` from breaking React / React Native prebuilt framework lookup
- Updated README and release notes

## Details

`1.2.0` injected `FRAMEWORK_SEARCH_PATHS` in the podspec for Simulator linking, but quoted `$(inherited)`:

```ruby
"$(inherited)"
```

Xcode then treated inherited paths as a single token, which could yield:

- `search path '...React-Core-prebuilt...' not found`
- `framework 'React' not found`

`1.2.1` uses the unquoted inherited form:

```ruby
$(inherited)
```

## Upgrade impact

- All iOS integrators should upgrade to `1.2.1`
- Android unchanged
- TS API and return shapes unchanged
- Simulator remains “builds, no real Bluetooth”
