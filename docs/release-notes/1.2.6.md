# 1.2.6

发布日期：待发布

## 变更摘要

- 升级包版本到 `1.2.6`
- iOS 连接状态事件与测量停止行为优化
- 同步更新文档入口与版本索引

## 框架改动

本次主要包含 iOS 原生层行为优化：

- 连接状态流转增强：
  - 增加/完善连接状态对外事件触发
  - 连接状态变化时对外状态映射更明确，减少业务层状态分叉处理
- 断开流程增强：
  - `disconnect` 分支中补充连接状态事件上报
  - 统一清理 `activeConnectDeviceId` / `activeMeasurementType`
- 测量停止流程优化：
  - `stopHeartRateTest` / `stopTemperatureTest` / `stopStressTest` / `stopBloodGlucoseTest`
  - 增加手动停止后的测量完成收口，统一 `manual_stop` 结束路径

## 版本同步

- `package.json` → `1.2.6`
- `module.json` → `1.2.6`
- `ios/VeepooSDK.podspec` → `1.2.6`

## 升级影响

- 不影响 Android
- 不改变公开 JS/TS API 签名
- 建议依赖连接状态事件和测量状态事件的业务升级到 `1.2.6`

## 发布前检查

建议发布前执行：

- `npm run typecheck`
- `npm run build`
- `npm run prepublishOnly`
- `npm pack --dry-run`
