# 1.2.6

**Released:** Pending

## Summary

- Bump package version to `1.2.6`
- iOS connection events and measurement stop behavior tightened
- Doc entry points and version index updated

## Native changes (iOS)

- Connection state:
  - More consistent outbound events
  - Clearer mapping when state changes
- Disconnect path:
  - Emits connection events from the `disconnect` branch
  - Clears `activeConnectDeviceId` / `activeMeasurementType`
- Stopping measurements:
  - `stopHeartRateTest` / `stopTemperatureTest` / `stopStressTest` / `stopBloodGlucoseTest`
  - Completes manual-stop flows with a unified `manual_stop` path

## Version alignment

- `package.json` → `1.2.6`
- `module.json` → `1.2.6`
- `ios/VeepooSDK.podspec` → `1.2.6`

## Upgrade impact

- Android unchanged
- Public JS/TS signatures unchanged
- Apps that depend on connection or measurement events should move to `1.2.6`

## Pre-release checklist

- `npm run typecheck`
- `npm run build`
- `npm run prepublishOnly`
- `npm pack --dry-run`
